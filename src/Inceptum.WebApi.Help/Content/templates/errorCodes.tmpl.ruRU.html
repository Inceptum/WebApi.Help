Для индикации результата выполнения запроса (успех или ошибка) API использует коды статуса HTTP протокола. В общем случае, коды из диапазона 2xx означают успешный запрос. Коды из диапазона 4xxx - ошибку запроса вызванную некорректными входными данными (например, неправильные параметры запроса). Коды из диапазона 5xx означают ошибку на стороне сервера.
Дополнительная информация об ошибке может быть передана в теле запроса.
